Making Localized Heater/Cooler For Agar Plates

For the experiments, we needed to heat or cool specific areas on agar plates, so we designed a system using Peltier devices as both heating and cooling sources. I created a circuit with Arduino to control the mode of the Peltier devices and to set them to specific temperatures. Additionally, I developed a Python UI application to manage these Peltier temperature regulations.
